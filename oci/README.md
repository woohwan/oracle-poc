### OCI PoC를 위한 구성

- ADB를 붙일 경우,  
wallet을 다운로드 받아서 home dir에 푼다.  
```  
pip install langchain-oracledb langchain-oci

mkdir -p ~/wallet_adb
unzip ~/Downloads/Wallet_*.zip -d ~/wallet_adb

```  

- 오라클 Vector DB에 임베딩할 경우,  
```  
pip install sentence-transformers
```  
```  
CREATE TABLE RICHARD_VECTORS (
    id     NUMBER GENERATED BY DEFAULT AS IDENTITY
           CONSTRAINT richard_vectors_pk PRIMARY KEY,
    text   CLOB,
    vector VECTOR(1536, FLOAT32) NOT NULL
);

CREATE VECTOR INDEX richard_hnsw_idx
ON richard_vectors(vector)
ORGANIZATION INMEMORY NEIGHBOR GRAPH
DISTANCE COSINE
WITH TARGET ACCURACY 90
PARAMETERS (
    type HNSW,
    neighbors 40,
    efconstruction 500
);

```
